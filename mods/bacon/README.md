## Install

###### Requirements

- [Minetest](http://www.minetest.net) >= 0.4.10
- default

[Download](https://github.com/davisonio/bacon/archive/master.zip) & unzip the mod, rename it to "bacon" and move it into your `minetest/mods` folder.
